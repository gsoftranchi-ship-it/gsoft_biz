# GSoft Biz ERP

A modern, multi-tenant ERP solution built with Flutter and Firebase, designed primarily for gyms, fitness centers, and small businesses.

---

## Features

### Core Modules
- Authentication
- Dashboard
- Member Management
- Attendance Management
- Product Management
- Billing & Invoicing
- Payment Management

### Billing
- Product Billing
- Member Billing
- Fixed Discount
- Percentage Discount
- Payment Status (Paid / Partial / Pending)
- Balance Calculation
- Invoice Summary
- Firestore Integration

### Technology Stack

- Flutter 3.38.x
- Dart
- Firebase Authentication
- Cloud Firestore
- Firebase Storage
- Firebase Cloud Messaging
- Provider State Management
